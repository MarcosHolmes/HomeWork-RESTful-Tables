# Project Title

Tarea | Tablas RESTful

## Getting Started/ Installing

Clonar el repositorio, abrir una terminal en la raíz del repositorio y ejecutar comando(s):

`npm run dev`| `npm start`

  #### Running (in postman)

  `http://localhost:3000/results/${n1}/${n2}`
